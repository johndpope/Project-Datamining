

How to run the Script:

1. uncomment the install.packages comments in the begining of the script if all the mentioned packages are not already installed.
2. Convert the retail, Game Of Thrones and Titanic dataset into .arff format.
	a) Before converting the Game Of thrones dataset, we need to remove the Name column from the dataset.
	b) Remove all the spaces and ' also.
	c) Then covert the Game Of Thrones Dataset to arff format.

3. set the correct path for the followign variable 
	datapath<-"//home//rkmalik//Desktop//allworkspaces//dm//project2//"
4. Put all the datasets at the datapath location.
5. Run the script.
